### 10.3 프로퍼티

객체 는 프로퍼티의 집합. 프로퍼티는 키와 값으로 구성

# 10-03

```javascript
var person = {
  // 프로퍼티 키는 name, 프로퍼티 값은 'Lee'
  name: "Lee",
  // 프로퍼티 키는 age, 프로퍼티 값은 20
  age: 20,
};
```
